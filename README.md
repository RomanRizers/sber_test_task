# Тестовые задания в СБЕР

## Задание №1 

Есть текст с произвольными символами. Нужно определить, корректно ли в нем
расположены квадратные скобки (правильно открыты и закрыты)
Пример с некорректными
“hello [ w]o[rld”
Пример с корректными
“hel[lo[worl]]d”
